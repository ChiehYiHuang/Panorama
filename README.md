# Panorama
Implement cube Panorama by using JAPanoView。

從立方全景圖像創建顯示360 - 180度全景，交互式平移和縮放。

(基於iOS Framework實現）


## Introduce ##
> Resource 檔案夾：專案中使用到的圖片兒。
> ThirdParty 檔案夾：第三方JAPanoView
> BrowserViewController : 實作全景圖。
